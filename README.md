# Paladin Front-End

Next.js application for Paladin triage system.

## Deployment

This project is configured for Vercel deployment. The Next.js app is located in the `triage-website/` directory.

### Local Development

```bash
cd triage-website
npm install
npm run dev
```

Visit `http://localhost:3000`

### Build

```bash
cd triage-website
npm run build
npm start
```

## Project Structure

- `triage-website/` - Main Next.js application
  - `app/` - Next.js App Router pages
  - `components/` - Reusable React components
  - `public/` - Static assets

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
